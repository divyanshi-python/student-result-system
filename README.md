# Student Result System

A Python mini project to manage student marks and grades.

## Features
- Add student names and marks
- Show topper
- Search A grade students
- Show total class strength
- Menu-based output
- Grade distribution
- Student search by name

## Concepts Used
- Python dictionary
- Loops
- Conditions
- Functions
- Menu-driven logic

## Future Improvements
